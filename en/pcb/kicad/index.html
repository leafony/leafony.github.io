KiCad Template Project

Overview

Section titled “Overview”

The Leafony template project for KiCad makes leaf board design a breeze.
The template project is configured with the following items

  • 2cm x 2cm Leaf outline
  • Leafony bus footprint
  • Design Rules.

Download

Section titled “Download”

You can download the template project from the link below.

GitHub - Leafony/cad-template

Development environment

Section titled “Development environment”
  • KiCad 5.0.0 or later

Usage

Section titled “Usage”

Create a new project

Section titled “Create a new project”
  1. Launch KiCad.
  2. File -> New -> Project -> Create Project from Template.
  3. Select the User Templates tab.
  4. See the following path from the template project downloaded under the template path.
  1. Select Leafony Leaf and click OK.

  1. Enter a project name and save the project to any location.
